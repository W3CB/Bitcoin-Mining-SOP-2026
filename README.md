## Overview

This repository contains **33 Standard Operating Procedures (SOPs)** for industrial Bitcoin mining operations, developed as the official companion to *The Ultimate Bitcoin Mining Handbook — 2026 Update*. Each SOP translates the handbook's guidance into executable, step-by-step procedures that mining operations can implement directly.

The library covers the full operational lifecycle — from hardware procurement and facility design through financial management, regulatory compliance, cybersecurity, and strategic exit planning.

### What's Included

| Type | Count | Description |
|------|-------|-------------|
| SOPs (PDF) | 33 | Step-by-step operational procedures |
| Excel Workbooks | 25 | Calculators, trackers, and decision tools |
| Supplementary Tools | 2 | Additional workbooks supporting multiple SOPs |

**Total:** 33 SOPs with 25 companion Excel tools across 8 operational categories.

Recommended implementation sequence:

1. **Pre-build:** SOP-006 → SOP-007 → SOP-008 → SOP-003
2. **Launch:** SOP-001 → SOP-011 → SOP-009 → SOP-010
3. **Financial setup:** SOP-012 → SOP-017 → SOP-014
4. **Security baseline:** SOP-022 → SOP-023 → SOP-025
5. **Compliance:** SOP-026 → SOP-029 → SOP-030
6. **Ongoing ops:** SOP-019 → SOP-020 → SOP-021

## Notes

- All Excel workbooks have been audited for formula integrity. Every file passes with zero errors.
- SOPs marked ⭐ are designated critical — they address the highest-impact operational and financial decisions and should be prioritized during initial implementation.
- The `supplementary/` folder contains tools that support multiple SOPs and are not exclusively tied to a single procedure number.
- SOP numbering runs 001–033. The sequence skips from SOP-021 to SOP-022 in the Mining Operations section as firmware, cybersecurity, and wallet security are numbered within their respective categories.
